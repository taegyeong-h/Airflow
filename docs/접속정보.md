# BDP 개발용 서버 접속정보

> [!Note]
> **서버 사양**
> - CPU : 2 Sockets x 8 Cores = Total 16 Cores **(32 vCPUs)**
> - 메모리 : 12 Slots x 16 GB = Total **192 GB**
> - 디스크 (host 시스템용 스토리지) SSD : 250 GB
> - 디스크 (VM용 스토리지) **SSD : 1 TB**
> - 디스크 (OS 이미지 다운로드용) HDD : 500 GB

> **※ vCPU, 메모리, (VM용)디스크 사이즈가 총량에 넘어가지 않게 주의!!!!**
